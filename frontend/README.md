# Начальная архитектура

## config

 - Конфигурация сборщика
 - Конфигурация env
 - Конфигурация https
 - Модули
 - Конфигурация тестов

## public

 - Статичные файлы
 - SEO-оптимизация

## scripts

 - Конфигурация скриптов (сборка, запуск, тесты)

## src

 - Входная точка приложения

(Осн. информация хранится в корневом ![README.md](../README.md))

# Запуск

```bash
cd ./AIFixed/frontend

npm install

npm start
```
